## 대주제 따릉이 운영 전략 제언

#### 참여자 : 이재원, 신지원, 이윤서, 최현, 황수민

## EDA 프로젝트 요약

### 1. 프로젝트 주제 및 목적

- **주제** : 서울시 공공자전거 '따릉이'의 이용 패턴 분석을 통한 운영 효율화 및 적자 개선 방안 제안
- **목적** : 늘어나는 따릉이 운영 적자 문제를 해결하기 위해 데이터 기반으로 이용 행태를 분석하고, 기상·시간대·연령·행정구역별 영향 요인을 정량화하여 대여소별 거치 전략을 최적화함으로써 운영 비용을 절감하고자 함

---

### 2. 데이터 전처리

**날씨 데이터**

- 기온은 비선형 관계 반영을 위해 평균 중심화 후 제곱 처리 (`기온_중심화_제곱`)
- 강수량은 연속형 수치 외 강수 여부(0/1) 이진 변수로 별도 생성
- 미세먼지(PM10)는 상관계수 약 0.04로 유의미한 관계가 없어 최종 회귀 모델에서 제외

**시간대 데이터**

- 민감도 분석을 통해 24개 시간 연속형 변수 대신 3구간 더미로 구조화
  - 출근(5-7시) / 퇴근(16-19시) / 기준시간(Non_Peak): 그 외
- 요일은 One-Hot Encoding 처리하여 평일/주말 패턴 차이 반영

**이용권 데이터**

- 전체 이용권을 정기권 / 일일권 / 기타(단체권·가족권 등)로 구분
- 기타 이용권은 전체의 1% 미만으로 분석에서 제외, 정기권·일일권 중심 분석

**연령대 데이터**

- 이용량 상위/하위 25개 정류장의 연령대 비율을 추출하여 비교 분석
- Mann-Whitney U test를 통해 그룹 간 연령 분포 차이 유의성 검정

**행정구역 데이터**

- 자치구별 총 대여건수 집계 및 연도별 시계열 시각화
- 지하철역과 따릉이 정거장 간 거리를 새로운 변수로 생성 (Standard Scaling 적용)
- 총생활인구 변수는 자치구별 인구 규모 편향 제거를 위해 Standard Scaling 처리

**종속변수**

- 대여건수에 로그 변환 적용(`log_rentals`)으로 왜도 제거 및 회귀 모델 정규성 확보

### 3. 분석 방법 및 결과

**날씨·기상 요인 분석**

- 기온은 약 22°C까지는 이용량과 비례 증가, 이후 감소하는 비선형 관계 확인
- 풍속·강수량 모두 이용량과 음의 상관관계를 보이며, 강수 여부에 따른 이용량 차이는 평균 약 30%에 달함
- 미세먼지(PM10) 농도와 대여건수 간에는 유의미한 상관관계 미확인 (상관계수 ≈ 0.04)

**이용권별 행태 분석**

- 전체 이용의 83.23%가 정기권, 15.91%가 일일권이며 연간 변동폭 ±0.3%p 미만으로 안정적
- 정기권 이용자 : 평균 이용시간 약 19.1분, 평균 이동거리 약 2,139m (단시간·단거리 통근 패턴)
- 일일권 이용자 : 평균 이용시간 약 29.4분, 평균 이동거리 약 3,437m (여가·관광 패턴)

**시간대 분석**

- 주중에는 출퇴근 시간대 대여가 집중되며 저녁 퇴근 피크가 두드러짐
- 주말은 낮 시간대에 이용이 분산되고 변동폭이 완만함
- 대여-반납 Net(반납-대여) 분석 결과, 주중 퇴근 시간대(18-23시)에 반납 과다 불균형이 심화됨
- 요일×시간대 조합 히트맵으로 불균형이 특정 요일의 특정 시간에 집중됨을 구조적으로 확인

**연령대 분석**

- 전체 이용자의 약 56%가 20-30대이며, 이용량 상위 25개 정류장은 20-40대 비중이 압도적
- 상위/하위 정류장의 연령 비율 차이에 대한 Mann-Whitney U test 결과 p-value 0.23 → 가설 기각, 연령 변수의 유의성 불확인
- 출퇴근과 따릉이 이용률의 상관관계 파악을 위해 지하철역과 따릉이 정거장 간 거리를 대안 변수로 제안

**행정구역 분석**

- 강서구·송파구·영등포구가 연간 이용량 최상위, 이용량 상위 대여소는 마곡나루역·잠실역 등 주요 지하철역 인근에 밀집
- 계절성은 자치구 공통으로 5-6월, 9월 최고 / 1-2월 최저

**다중선형회귀 분석**

- 복합적인 변수들의 개별 영향력을 정량적으로 분리하기 위해 다중선형회귀 채택
- 이용량을 가장 크게 감소시키는 요인 : 강수 여부 (계수 약 -0.25~-0.28)
- 이용량에 가장 긍정적인 영향 : 저녁 피크타임 (계수 약 +0.20~+0.21)
- 주말·공휴일은 모두 음의 계수 → 따릉이는 여가보다 평일 출퇴근 수단으로 더 활발히 이용됨
- 자치구별 분석에서 강남구 대비 동작구(+0.30↑), 마포구(+0.18↑)는 높은 기본 수요, 관악구(-0.19~-0.20), 서초구(-0.10~-0.14)는 낮은 수요 기록 (3개년 일관)

**계층적 군집 분석 (K=4, 실루엣 점수 0.58)**

| 군집 | 성격 | 해당 자치구 | 운영 전략 |
|:---:|:---:|:---|:---|
| Cluster 1 | 효율형 | 동작구 | 거치대 부족 방지, 실시간 재배치 우선순위 부여 |
| Cluster 2 | 성장형 | 마포, 성동, 영등포, 종로 등 | 퇴근·여가 시간대 수요 중심 배치 |
| Cluster 3 | 정체형 | 강서, 송파, 노원, 중구 등 | 이용률 저조 대여소 점검 선행 |
| Cluster 4 | 취약형 | 관악, 서초, 양천, 중랑 | 지형적 원인 추가 분석 후 배치 전략 고안 |

### 4. 결론

따릉이 이용량에는 기상 조건(강수 여부·기온), 시간대(평일 퇴근 피크), 공간적 특성(역세권 접근성·자치구)이 복합적으로 작용하며, 강수 시 이용량이 약 30% 급감하고 저녁 퇴근 시간대 수요가 가장 높게 나타났다. 이용자의 83%가 정기권을 사용하는 고정 수요 기반 서비스임을 고려할 때, 따릉이 배치 문제는 단순 운영 이슈가 아닌 구조적인 시간·공간적 불균형 문제로 이해해야 한다. 자치구를 4개 군집으로 분류하여 효율형·성장형·정체형·취약형에 맞는 차별화된 배치 전략을 도입함으로써 운영 효율화와 실질적인 적자 관리가 가능할 것으로 제언한다.

---

### 5. 아쉬운 점

- **공간 단위의 거시성** : 분석 단위가 '자치구'로 설정되어 같은 구 내에서도 대여소별 고도·주변 도로 환경 차이가 구 평균값으로 뭉뚱그려지는 한계가 존재함
- **미수집 변수의 간접 추론** : 지형 경사도, 자전거 도로 연속성, 지하철역 승하차 인원수 등 이용량에 직접 영향을 줄 수 있는 변수들을 포함하지 못하고 자치구 더미 변수를 통해 간접 추론하는 데 그침

---

### 6. 추가로 하면 좋을 분석 방법

- **비선형 머신러닝 모델 적용** : 외부 데이터(경사도, 자전거 도로 정보, 지하철 승하차량 등)를 추가하고 Random Forest, XGBoost 등 비선형 모델을 결합하여 더욱 정밀한 실시간 재배치 알고리즘으로 발전시킬 수 있음
- **대여소 단위 미시 분석** : 자치구 수준이 아닌 개별 대여소 단위의 수요 예측 모델을 구축하여 대여소별 최적 거치 수량 산출
- **지하철역 승하차량 연계 분석** : 제안한 '지하철역-대여소 거리' 변수에 실제 승하차량 데이터를 결합해 역세권 수요 연동성을 정량화
- **따릉이 경쟁 서비스 비교 분석** : 킥보드·전동자전거 등 대체 교통수단 이용 데이터와 비교하여 2024년 이후 이용량 감소 원인을 규명
